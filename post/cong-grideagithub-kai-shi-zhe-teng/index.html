从Gridea+Github开始折腾

2021-11-25 3 min read

一直想找个地方，整理自己的心得，不至于每次都要去度娘重新再找。
可是实在太懒了，那些博客平台又觉得不放心，哪天倒了自己的心血也就没了。
无意间搜GMT，看到具神也将博客转到github上了。
可是注册github后，水平有限，发现好乱，于是又去看其他博客平台了，
csdn、简书、博客园、知乎、掘金、思否、开源中国……
还去知乎看了大神的推荐和对比https://www.zhihu.com/question/21739407。
选择困难症的我犹豫了好几天后就此搁置了。
突然有一天，从某公众号看到Gridea+Github的介绍很吸引人。
决定开始折腾下。
具体公众号搜不到了，百度一搜gridea+Github一堆教程。
其实只要去Gridea官网看一下简明教程就好了。
这里只记录下遇到的坑，确实很坑。
折腾了半天，发现Gridea界面没法远程连接，按键全部是灰色的？
以为是没装GIT缘故，装了之后问题依旧……
换了机器也是如此，心灰意冷，网上也找不到答案，于是又去寻找好的博客平台去……
转了一圈，还是觉得不合适，心中那股倔劲儿又上来了，
终于搜到https://www.bilibili.com/read/cv13059298 作者：第十三只黑鸟这个文章后，
心想会不会也是这个低级错误，一试果然是……
“复制信息到远程连接的基础设置里去的时候！！！！！！多了空格！！！！！！
这个真的是最坑的，小白很容易被误伤，而且肉眼看不出）”
满心欢喜的填好点保存，再点远程连接，问题又来了，连接失败……
找了好多办法，包括但不限于https://zhuanlan.zhihu.com/p/106781771 这篇博文类似的方法
于是又去寻找好的博客平台去……
寻找未果，又过了几天，
终于发现了作者闷声的干货https://zhuanlan.zhihu.com/p/406234933
让我这个小白心生幻想，重燃希望
安装了作者修改的gridea版本，可以远程连接成功了，哈哈，但是依然无法同步……
又一盆水浇灭了我心中小火，
于是又去寻找好的博客平台去……
时间又过去几天，心想不能再拖了，
有些东西想整理的又忘了，随便找个博客得了，反正也是记录给自己看的。
于是再试最后一次，我又找出作者闷声的干货文，仔细阅读了下，发现是不是没装GIT和dev-sidecar
重新安装后，再试，我靠，行了，这次是同步成功了！
于是……又不知道写啥了。
时间又过去几天，
就先把这次折腾的经理记下来吧，
小白一枚，万恶的坑，
最后还是要感谢闷声大神的博文的指导！
终于可以开始自己的博客记录了。
发这条博客同步又出现问题，
报错SSL_read: Connection was reset, errno 10054
搜索后借鉴https://blog.csdn.net/qq_29493173/article/details/114534057
第三种办法解决
git config --global http.sslVerify "false"
